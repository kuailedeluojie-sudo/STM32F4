【*】程序简介

-工程名称：RAM调试―多彩流水灯
-实验平台: 野火STM32 F429 开发板 
-MDK版本：5.16
-ST固件库版本：1.5.1


【 ！】功能简介：
 学习sct文件应用，学会使用RAM调试方式

【 ！！】注意事项：
 本工程支持RAM调试方式，具体见《零死角玩转STM32》中对应的章节说明
 
 使用RAM调试时，需要把开发板左侧引出的BOOT0、BOOT1排针使用跳线帽连接到3.3V
	BOOT0<--->3.3
	BOOT1<--->3.3
 

/*********************************************************************************************/

	我们用于清理MDK杂项文件的keilkill.bat工具可能会删除本工程的sct文件，
	若在Output目录下找不到sct文件，请新建一个名为“SRAM_调试.sct”的文件到该目录，
	并复制如下内容到该文件：

/*-----------------sct文件内容，不要复制这一行------------------------*/

; *************************************************************
; *** Scatter-Loading Description File generated by uVision ***
; *************************************************************

LR_IROM1 0x20000000 0x00010000  {    ; load region size_region
  ER_IROM1 0x20000000 0x00010000  {  ; load address = execution address
   *.o (RESET, +First)
   *(InRoot$$Sections)
   .ANY (+RO)
  }
  RW_IRAM1 0x20010000 0x00020000  {  ; RW data
   .ANY (+RW +ZI)
  }
}
/*-----------------不要复制这一行-----------------*/

也可打开工程目录下的“sct文件备份.txt”文件，复制其内容

/*********************************************************************************************/
【*】 引脚分配

LED：
RGB彩灯的三个阴极接到STM32三个引脚上，LED低电平亮。

	R<--->PH10
	G<--->PH11
	B<--->PH12
	
	LED4<--->PD11
	
/*********************************************************************************************/

【*】 时钟

A.晶振：
-外部高速晶振：25MHz
-RTC晶振：32.768KHz

B.各总线运行时钟：
-系统时钟 = SYCCLK = AHB1 = 180MHz
-APB2 = 90MHz 
-APB1 = 45MHz

C.浮点运算单元：
  不使用
